# Vetted Safe Test Assets for Local Upscaling

This version keeps only assets hosted on reputable **HTTPS** websites and removes sources that could raise security or provenance concerns. The remaining links point to well-known organizations or platforms with documented licensing terms, including Blender, NASA, Unsplash, Pexels, and LibriVox.[web:154][page:0][page:1]

## Vetted source standards

Each source below meets these baseline checks:

- Uses **HTTPS** rather than insecure HTTP.[page:0][page:1]
- Is hosted by a recognizable primary publisher or platform, such as Blender, NASA, Unsplash, Pexels, or LibriVox.[web:154][web:224][web:200][page:0][page:1]
- Has documented reuse terms or public-domain status available from the publisher.[web:210][page:0][web:221][page:1]

## Download safety notes

Even when a source is reputable and uses HTTPS, the safest workflow is to download only from the official page, avoid browser extensions or third-party downloaders, and scan files locally after download if desired. HTTPS and publisher reputation reduce risk, but they do not guarantee that every downstream action on a local machine is risk-free.[page:0][page:1]
